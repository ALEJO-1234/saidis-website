# Saidis Acessórios - Website

Uma loja virtual elegante para venda de pulseiras artesanais e acessórios únicos.

## Características

### 🎨 Design
- Interface moderna e responsiva
- Cores suaves inspiradas na marca (roxo, rosa, amarelo)
- Design mobile-first
- Animações e transições fluidas

### 🛒 Funcionalidades
- **Carrinho de Compras**: Adicione, remova e modifique quantidades
- **Sistema de Favoritos**: Marque produtos como favoritos
- **Busca**: Pesquise por produtos (simulação)
- **Newsletter**: Inscreva-se para receber novidades
- **Responsive**: Funciona perfeitamente em desktop, tablet e mobile

### 🌟 Produtos em Destaque
- Pulseira Flor de Lótus
- Pulseira Cristais Azuis  
- Pulseira Borboleta Dourada
- Pulseira Pedras Naturais

### 📦 Benefícios da Loja
- ✈️ Entrega para todo o Brasil
- 💳 Pagamento seguro (PIX, Cartão)
- 🔄 30 dias para trocas
- 📞 Suporte ao cliente

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com Flexbox e Grid
- **JavaScript**: Interatividade e funcionalidades
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia (Montserrat)

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Navegue pela loja e teste as funcionalidades:
   - Adicione produtos ao carrinho
   - Marque favoritos
   - Use a busca
   - Inscreva-se na newsletter
   - Teste a responsividade redimensionando a janela

## Deploy

Este site está configurado para ser facilmente deployado em:
- **GitHub Pages**: Basta fazer push para um repositório GitHub
- **Netlify**: Arraste a pasta para o Netlify Drop
- **Vercel**: Conecte o repositório GitHub
- **Cualquer servidor web**: Upload dos arquivos HTML
